# Roamsphere
RoamSphere is a **demo travel website** built as a front-end project.  
It presents destinations, travel packages, brand information, and a contact flow in a clean multi-page layout.

## Overview

This project focuses on:

- responsive page structure
- consistent navigation and footer across pages
- simple interactive behaviors with JavaScript
- clear visual hierarchy using Bootstrap + custom CSS

## Pages

- `pages/index.html` - home page and featured sections
- `pages/destinations.html` - destination cards with filtering
- `pages/packages.html` - package cards with expandable list
- `pages/about.html` - brand story, values, and testimonials
- `pages/contact.html` - contact form and business details

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript
- Bootstrap Icons

## Run Locally

Open `pages/index.html` in a browser.

For a smoother workflow, run the folder with a static server (for example, VS Code Live Server).

## Notes

- This is a demo project; it does not connect to real booking or payment systems.
- Form actions are illustrative and handled on the front end.
